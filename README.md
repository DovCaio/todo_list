# 📝 To-do App com Angular e Springboot

Uma aplicação simples de controle de tarefas, construída com Angular no front-end e Springboot no back-end.

---

## 🎯 Propósito

Este projeto tem como objetivo servir como base de aprendizado para quem está começando com:
- Angular no front-end
- Integração entre front-end e back-end usando APIs REST

---

## 🚀 Tecnologias Utilizadas

### Back-end
- Springboot

### Front-end
- Angular 16+ (CLI)
- Angular HttpClient

---

## ⚙️ Funcionalidades

- Criar tarefas
- Listar tarefas
- Marcar como concluída
- Deletar tarefas

Para deixar mais complexo:

Cadastro, edição, exclusão e listagem de tarefas.

Organização das tarefas em projetos (cada projeto tem várias tarefas).

Tarefas com: título, descrição, prioridade, data de vencimento, status (pendente, em progresso, concluída), etiquetas (tags).

Subtarefas: uma tarefa pode ter várias subtarefas aninhadas.

Comentários e anexos em cada tarefa.

Notificações (por email ou push) para prazos próximos.

Filtros e busca avançada por múltiplos critérios (data, status, prioridade, projeto, etiquetas).

Sistema de usuários com autenticação, perfis e permissões (ex.: somente donos do projeto podem editar).

Histórico de alterações (audit trail) para cada tarefa.

Integração com calendário (Google Calendar, Outlook).

Versão offline com sincronização quando online (PWA).
